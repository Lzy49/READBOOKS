## 大脑运行原理
1. 节能 大脑是老年人本着能省就省的原则过日子。具体省在
  1. 节省注意力 ： 我们在只会注意到我们感兴趣或者重要的事情。其他一些环境例如周围同事在做什么其实我们并没有关注到
  2. 节省理性：我们会因为理性和直觉来做决定。但是遇到大多数事情我们会第一时间使用直觉而不是理性，例如 看到一个程序员第一反应就是他比较内向，不善沟通。
  3. 习惯：为了节省大脑会把很多需要思考的事情变成习惯，这样就不需要思考就能完成。比如开车，打字。都是从一个理性考虑的事情变成了习惯。
2. 追求稳定 大脑因为喜欢过节能的日子，所以追求稳定，因为稳定不需要那么多思考。主要体现在：
  1. 喜欢确定性：大脑不喜欢未知，喜欢确定，哪怕是一个坏的结果。比如 公司在裁员下一个可能是你，听到这里就会惶惶不安。而明确告诉你下一个就是你，就不会不安，而去处理。
  2. 一致性：大脑不喜欢认错，所以一旦有和自己想的不一样的情况就会给自己找台阶：比如 认为某明星是个正经人，当他犯了事我们会认为他是被小人所害。
  3. 适应性：大脑不喜欢变来变去，大脑喜欢一如既往的相同。比如每天11点睡觉。而不是 今天 10点明天1点，后天11点这种就很难习惯。
3. 预测 在事情没发生前我们往往会根据自己的想法去预测。去想想。而且生活中的每一刻我们都在预测。
   1. 大脑通过预测了解世界：我们生活的每一瞬间都会为大脑编辑一条规则。而大脑会通过这些规则去预测下一秒下一个场景会发生的事情。例如 第一节课老师说要起立问好，那下一节课我们会下意识的去起立问好。
   2. 我们的世界其实是个幻象：世界其实是大脑播放给我们看的世界，只不过它和真正的世界几乎一模一样，但是有一些地方可能会被我们忽略例如：一行字顺序是错误的。但是我们可以自动的忽略掉。
   3. 一举一动都在训练大脑：我们的做的每一件事都会为大脑创建一条规则，从而优化我们的大脑。所以，如果我们一直做简单的事情，或重复的事，那大脑不会有太大的进步。
   4. 大脑预判了我们的预判：如果一直做简单的事情，大脑一直知道下一秒要做什么，会发生什么，这个时候我们会觉得无聊需要寻找刺激，原因其实是因为如果我们的祖先一直在一个不变的环境下是无法生存的。所以积累了这一条规则。且因为这条规则延续至今。
   5. 如何训练我们的大脑：因为我们的大脑又喜欢稳定，又不能太过稳定，所以我们应该学一些我们熟悉，但是又有一些不会的东西（舒适区）。
4. 反馈：人是趋利避害的，弊害其实表现在 节能上趋利表现在反馈上。
   1. 反馈的底层其实是多巴胺的学习和强化机制。如果做一个事情有正反馈，则我们会很开心也会有更大的动力去做。例如得到某个人经常夸奖你，你就很容易为他做更多事情。因为夸奖会满足我们的自信心会刺激大脑。
   2. 快乐和动机并不是一个东西。其实多巴胺给我们带来的不是快乐，而是动机。快乐只有达到某个点才会存在。而动机会促使我们靠近哪个点。
   3. 大脑的短视：获取多巴胺是有一套独立的机制的，我们更容易选择难度低获得少的那条路走。例如 找业务才能拿到高工资，不找业务可以拿到底薪。我们更愿意去拿底薪而放弃找业务。
   4. 大脑短视的秘密：其实我们获取的多巴胺并不是奖励本身，而是我们奖励本身 减去 我们想象的奖励。这个差值越大，就会提供更多的动力来支持我们做这件事情。所以为了得到更大的目标我们应该将任务分成小目标，来完成一个有一个的小奖励（或许小奖励还有有意外的奖励）如此去刺激我们产生更多的动力，去完成小目标。在这个过程中得到我们想要的提升。最后的大奖励也会给我们更多的动力。
## 情绪
1. 情绪是我们的一个报警器：
   1. 情绪其实依然是为了服务我们，保护我们。它想在合适的场景为我们提供更多的能量。但因为世界的变迁，超出了我们的进化。所以情绪变的有些跟不上我们生活的脚步。
   2. 如果我们马上要交答卷，则情绪会非常的紧张，这份紧张会让我们集中精神，去完成答卷。而有的时候，我们会因为某些事情而愤怒，这种愤怒可能是不必要的。但是情绪它也只是想保护我们在当时不受到伤害。
2. 愤怒恐惧焦虑
   1. 愤怒恐惧焦虑3种情绪是因为 有 威胁 才会报警的一种报警机制。
      1. 愤怒 ：发生已知负面的事情 + 可控制
      2. 恐惧 ：发生已知负面的事情 + 不可控
      3. 焦虑 ：发生未知负面的事情 + 不可控
   2. 与负面情绪交流：
      1. 倾听 ：情绪它是最爱我们的，是一个敏感的小孩子。如果他来了，说明我们一定遇到了一些威胁，我们不妨听听他的声音比如问问：因为什么有了这个情绪。哪里有了风吹草动。
      2. 思考 ：情绪出现时，可能只是他太敏感了。因为世界上 85%的坏事都不会发生而剩下的15%中75%远远不如我们想的糟糕。所以我们没必要为他们太过担心，当然我们也应该做一些准备。以防不测
      3. 安抚 ：情绪它来了，它非常紧张，我们要安抚他。和他说 ：好的我知道了，我想过了这件事情思考过了。你真棒，谢谢你。不用担心。
   3. 情绪是一个参考，而决策还是要我们自己去做的。不要因为情绪影响决策。
   4. 和情绪讲道理让他长大不要这么敏感
      1. 变的坚强：一件事往往有多个角度。如果一件事发生了，有情绪了。我们应该去分析下这件事情。并且从对我们最有利的方向想一下。例如某人说这件事这么简单，我去弄吧，你还是别弄了。我们可能会以为他觉得我们差，但其实是他觉得顺手就能搞定。所以把事情想的好一点，大家都是善良的。
3. 日常小手段，让自己变的更强。情绪更稳定。
   1. 情绪日记本：记录自己发生情绪的瞬间。
      1. 找一个时间去记录自己发生情绪的瞬间（每天，每周）
      2. 情绪分类
         1. 烦躁的事
         2. 焦虑的事
         3. 快乐的事
         4. 感恩的事
      3. 是什么让我们产生了这种情绪：原因是，我们是怎么想的。我们想的对吗
      4. 我有什么感觉，我有什么举动
      5. 如果我们的朋友发生了同样的事情，我会怎么劝说他。
      6. 如果下次遇到这件事我应该怎么做。这么做的好处是什么
   2. 焦虑收纳箱：
      1. 当我们有焦虑的感觉时，请第一时间把它记下来。并在今天处理情绪日记的时候处理。
      2. 焦虑产生 -> 记录 -> 告诉自己先别管那么多 -> 晚上再说
   3. 后花园：
      1. 找一些自己喜欢干的事情，并把它做的不错。
      2. 好处是建立了自信心，让自己不那么无聊而产生焦虑
   4. 社交：多和别人聊聊天，吃吃饭。聊天让人开心
   5. 阳光：没事出去晒晒太阳，看看世界，挺好的。
   6. 睡眠：睡好才能有力气面对困难。
   7. 锻炼：强健身体，身体是革命的本钱
   8. 多读书，思考：强健认知，多多锻炼自己的脑袋，让他多多思考。成为一个更强的大哥保护情绪。
   9. 认识到管理情绪的好处：我们对情绪的安抚，思考。也会成为我们的一种能力，这种能力让我们面对困难，能够更加冷静。让我与别人相处时，可以更加敏感的去想别人的情绪。从而去帮助别人。且得到更多的好感。
   10. 主动的去做更多的决策：从小事做起，建立自己决策能力。让我们更好的去管理情绪。管理拖延
### 自我驱动：如何长时间坚持做一件事
1. 自制力是无限的（实践证明）。动力也是无限的，笃定自己可以完成。就一定可以完成。
2. 为自己想做的事情建立系统
   1. 目标结果：这件事做成以后我会得到什么。我为了什么而做这件事情。
   2. 如何做：我如何做才能离这个结果近，这件事难实现的点是什么。我每天做点什么可以解决这个困难
   3. 每天做：每天去做第二步总结出的事情。并问问自己今天有没有离目标更近，如果没有原因出在哪里，需不需要改变。如果更近了，有没有很开心。
   4. 继续维护这套系统。并在自己觉得没有离目标更近的时候，优化整个系统。
3. 注意点
   1. 不要做无意义的努力，它只会带来负面的感觉，要学就真的学会。
   2. 如果坚持的结果与预设的结果没有缩小差距，那一定是系统出现了问题。不是我们能力出现了问题。我们要去想想第二步是不是有错误。
   3. 万事开头难，一开始我们是要改变一些大脑喜欢的稳定性的。所以，给自己一个小目标比如坚持10分钟。但是依然要问自己这10分钟有意义吗，学到了什么。
### 果断行动：如何摆脱懒惰，拖延和畏难。
首先拖延的原因是因为 动力不足 和 阻力太大。展开分析
- 动力不足
  - 原因：动力不足的原因就是没有多巴胺，就是做完以后没有强反馈。分为两个点：
    - 内因：
      - 根源：没有准备好。这是我们没有动力做的一个最大原因，我们往往想要面面俱到，但实际上根本不可能。比如学英语就要找一个超级无敌好的教程，一个超级无敌好的规则。这完全不可能的。
      - 解决：
        - 了解核心：不做后悔一辈子，做错后悔一会会。我们往往会记得自己没有做什么导致的什么。而记不得做错什么而导致什么。因为错了也有结果了，甘心了。但是不做，会不甘心的。
        - 挑战困难：一件苦难的事，我们要想想我们为什么想做。然后去尝试一下，如果不尝试，靠想是永远实现不了的。即使错了，也可以改。
        - 强大内我：多做决定，只要有机会就做决定。立刻马上做决定。我们每一次决定都是对自己的肯定，不要怕错。
        - 多做困难的事：每一次做困难的事都是对自己的一次磨砺。我们会记住它，并在下一个困难来临时笑笑说，小问题。不如我上次难。
        - 多想后果，解决后果：其实做准备也是有益的，但是这个准备必须是有用的。我们只要一个可以完成目标，或者处理某些问题的准备，而不是拖延。
    - 外因：
      - 根源：我们定的目标是有结果的有好处的，但是它一定是要努力以后才能得到的。但是我们生活中有很多不需要努力就能得到的奖励比如玩手机，刷视频。它们会分散我们的注意力，而且如果这个目标的奖励获取的周期越长，那分散注意力的时候就越多,就越是看不到完成的希望。那就越来越没动力了。得到公式：动力 = （完成任务的信心 * 得到的好处）/ (分心 * 得到的周期)。
      - 解决：要解决上面的问题其实就是把 分心 和 得到周期 缩短 ， 把 信心 和 好处 拉满。
        - 增加信心，减少周期：把一个长期的任务分成一个个短期的任务。这个短期的任务必须是可以完成的，且一定要有一个靠近最终结果的奖励而不是另一个不相关的奖励。比如：想要学会英语。那本周的任务就是新概念 1 - 5 章节 ， 但是结果不是 1 - 5 章节打卡完毕，而是 1 - 5章节学会了什么，这些东西对我学英语起到了什么作用。
        - 增加信心，增加得到的好处：锻炼自己，让自己做事情更有动力：
          - 找自己感兴趣又有难度的事情，并完成它 增加信心
          - 在下一次做同类型的任务时，掐下时间，和上次比一比。看看自己进步多少
          - 给自己提高一点的目标，并去努力完成它，看看自己有多强。
        - 增加信心：要相信，每一次的完成与未完成只要去努力了，都是有经验的，不要怕失败。努力去完成，即使这次没有完美完成，但是下次一定会做的更好。
        - 减少分心：
          - 和自己说，努力1小时，这1小时啥都不干，就干一件事。并把分心的东西放远
          - 给自己定 deadline 逼自己一把
- 阻力太大
  - 原因：阻力分为两种：
    - 内在阻力：
      - 根源：内在的阻力产生在我们本来就喜欢稳定，我们想要做一个事情时会破坏这个稳定的小程序。
      - 解决：
        - 先改变一点点：比如每天做一个10分钟
        - 事件替换：记录下最近一周每个时间点做的事情，把没有必要的事情划掉，填上自己想做的事情。
    - 外在阻力：
      - 根源：外在的阻力在于我们不了解，或者说，因为不了解产生了很多幻想，而这些幻想往往只有那些负面的会被我们记住，从而产生阻力。例如：我们想要出去玩，这个时候我们想到某一次出去玩找了2小时车位最后没找到，产生找车位的恐惧，然后就没出去。但其实我们上次，上上次出去玩的时候压根没发生找车位没找到这种事。
      - 解决：想象其实是我们默认的一个状态，没事做就会走神，就会想东西。记住让我们犯难的瞬间也是脑子趋利避害的一个机制。它就会记住伤害过它的瞬间。
        - 遇到困难，勇敢的面对困难：记录一下困难，并且用智慧的大脑问问自己，这事发生的概率有多大。出现了我能解决吗
        - 锻炼专注：正念
          - 缓慢呼吸10秒呼吸一次，坚持 15分钟
          - 随时随地闭上眼睛，感受下自己的四肢或者问问自己现在在想什么。
        - 主动想有用的东西：我今天学了啥，做了啥，有啥开心事，有啥烦恼。
        - 做就完事了：先做一下试试，不行咱再改。
### 重塑知识：在脑海中画一张地图。
1. 纠正现在的观点
   1. 读书的时明白作者想要表达的内容，只是第一步。
   2. 学习不应该是被动的而是主动的
   3. 接触新课题的最好方式是找一个东西先试试，但学习有基础的东西是解决一个个问题。
   4. 学东西不是学简单，而是学明白。
2. 如何正确学习
   1. 正确的学习原理是：让未知和已知联系到一起
   2. 了解：学习一个未知的内容应该从了解开始，这里是一些被动的获取，被动的收集。结果会收获一些信息，一些数据。
   3. 整理：找一个时间，整理这段时间的所学。并将相关的一个主题的内容收容到一起。并分析：
      1. 他们的联系是什么 -> 他们是一类信息，他们是因果关系，
      2. 他们的共同点是什么
      3. 他们的不同点是什么
   4. 想想：关于这个题目，我还有什么不明白。或者不太清楚。
      1. 这些观点和我有什么联系
      2. 我在生活中能怎么使用这个观点提速
      3. 如果我要讲给别人，我应该怎么讲
   5. 玩玩：
      1. 和别人聊聊这个观点，
      2. 利用这个观点在生活中试试。
